# Layer-wise Overlap Visualization on Wan2.1-1.3B

We use Wan2.1-14B as the teacher model to characterize global token importance, which is distilled into the AR model. Importantly, the 1.3B model exhibits qualitatively similar attention patterns, demonstrating that the observed behaviors are consistent across model scales.
